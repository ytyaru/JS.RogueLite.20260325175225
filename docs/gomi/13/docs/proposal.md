# 企画書: 10秒の意思決定ゲーム (仮)

## 1. コンセプト
「絵も音もない、だが脳が熱くなる10秒間」
最小限の要素で、じゃんけんやダイス投げ以上の「駆け引き」を体験させる、超短期的ローグライト・バトル。

## 2. ターゲット
- 隙間時間に、頭を少しだけ使いたい人。
- 広告や登録に疲れた、純粋な「玩具」を求める人。
- 派手な演出よりも、論理的なゲームメカニクスを好む人。

## 3. ゲームの概要
- **対戦形式**: 1vs1のターン制バトル。
- **リソース**: 4種類の札（斬る、捨て身、薬、自然治癒）。
- **勝利条件**: 相手の「命（♥）」を0にする。
- **特徴**: 「最大ライフ（♡）」の削り合いと、連続選択による累積ダメージが、短時間での逆転劇を生む。
